# Retail Sales & Profit Analysis Dashboard (Excel)

## Project Overview
This project analyzes retail sales and profit data using Microsoft Excel to support data-driven business decisions.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Power Query
- Charts & Slicers

## Key Analysis Performed
- Sales and profit analysis by product category
- Regional performance comparison (North, South, East, West)
- Payment method analysis (Cash, Card, EMI)
- Identification of high-revenue and high-margin products

## Key Insights
- Home Appliances generated the highest profit margin (25%).
- EMI payments significantly increased average order value.
- North and West regions showed strong sales performance.

## Files Included
- `Retail_Sales_Profit_Analysis.xlsx` – Excel dashboard file
- `dashboard_screenshot.png` – Dashboard preview

## How to Use
Open the Excel file and use slicers to filter by category, region, and payment method.
